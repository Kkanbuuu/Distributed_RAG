# Scripts Directory

This directory contains all shell scripts for the Distributed RAG project.

## Development Scripts

### `setup-venv.sh`
Sets up the unified Python virtual environment for local development.

```bash
./scripts/setup-venv.sh
```

### Run Scripts
Scripts to run all generator, orchestrator, retriever services collectively in venv

- `run-all.sh` - Run all services concurrently

```bash
./scripts/run-all.sh
```

## Usage

All scripts are executable and include help text. Run any script without arguments to see usage information.
